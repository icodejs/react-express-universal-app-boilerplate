# React Universal App Boilerplate

React client-side and server-side rendering using React, React-Engine and Express.

## Setup
```
$ npm install
$ gulp
$ npm start

# Visit: http://localhost:5000
```

## Directory structure
```
├── bin
│   └── www
├── gulpfile.js
├── package.json
├── public
│   ├── app.js
│   ├── dist
│   │   └── bundle.js
│   └── js
│       ├── components
│       │   └── head.react.js
│       └── views
│           ├── home.js
│           └── layout.js
└── server.js
```
